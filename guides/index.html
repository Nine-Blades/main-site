Digital Resources

Easygard

A mobile friendly web site that allows easy access to Amtgard V8 Rules, Spells, Classes, Abilities, Battlegames, and more.

mORK - Mobile ORK for Amtgard

mORK is an iOS and Android app that allows players to see a summary of their character and also allows for QR code, NFC, or Player search to add credits.

jsork - Amtgard ORK Reports

jsork is a set of JavaScript APIs that talk to the Amtgard ORK. Using these APIs there are many reports that you can find on this site.

Spellbound Tome Web and Mobile App

Spellbound is a Website and mobile app for Amtgard rules, spell lists, and much more.

Swiftgard Spell List App

Simple, quick and easy to use. Spell reference and list creator for all Amtgard classes. Also available on Google Play.

Sappy Spellbook

An unofficial tool for creating Amtgard spell lists and a quick reference for Martial Class abilities.
